# Hướng dẫn nhanh

## Cài đặt và chạy

```bash
# 1. Cài đặt dependencies
npm install

# 2. Chạy development server
npm run dev
```

Mở trình duyệt tại: http://localhost:3000

## Các trang có sẵn

- `/` - Trang chủ (Home/Newsfeed)
- `/login` - Đăng nhập
- `/signup` - Đăng ký
- `/profile` - Hồ sơ cá nhân
- `/messages` - Tin nhắn
- `/notifications` - Thông báo
- `/settings` - Cài đặt
- `/friends` - Danh sách bạn bè
- `/favorites` - Bài viết yêu thích
- `/saved` - Bài viết đã lưu
- `/trending` - Xu hướng

## Tính năng chính

✅ Dark mode (chuyển đổi bằng icon mặt trăng/mặt trời)
✅ Responsive design (mobile-first)
✅ Animations mượt mà với Framer Motion
✅ Glassmorphism effects
✅ Component-based architecture

## Lưu ý

- Dự án sử dụng dummy data để demo UI
- Tất cả hình ảnh sử dụng placeholder services (pravatar.cc, picsum.photos)
- Dark mode được lưu trong localStorage
